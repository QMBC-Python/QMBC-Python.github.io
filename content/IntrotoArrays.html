1.4 - Intro to Arrays¶

Video: Intro to Arrays

Video available at https://youtube.com/watch?v=-nqZEtNz6oY

What is Numpy?

NumPy = Numerical Python

Core library for scientific computing

Most “Matlab-y” data type

Lots of documentation: https://numpy.org/doc/stable/reference/

Lots of tutorials including: https://cs231n.github.io/python-numpy-tutorial/

import numpy as np

Array overview¶

Arrays are a new data type!

x = np.array([5, 1, 2])

x

array([5, 1, 2])


x = np.array([[3, 2], [1, 4]])

print(x)

[[3 2]
 [1 4]]


type(x)

numpy.ndarray

x.dtype

dtype('int64')

x = np.array([[3.1, 2], [1.2, 4]])

x.dtype

dtype('float64')

You can manually set the data type during array creation.

x = np.array([[3.1, 2], [1.2, 4]], dtype = 'int')

print(x)

[[3 2]
 [1 4]]

You can also change the data type later

x = x.astype('float')

x.dtype

dtype('float64')

x.shape

(2, 2)

Creating arrays¶

There are many ways to intialize arrays. Let’s start with the basics

  1. Initialize with numbers in closed brackets (python lists)

x = np.array([4, 6, 1])

x

array([4, 6, 1])

x = np.array([[4, 6, 1], [3, 4, 5]])

print(x)
print(x.shape)

[[4 6 1]
 [3 4 5]]
(2, 3)

  1. Initialize with zeros or ones

x = np.zeros(shape = (2, 2, 4, 8, 10, 2))

x.shape

(2, 2, 4, 8, 10, 2)

x = np.ones(shape = (5, 1))

  1. Initialize with evenly space numbers

Given interval and step size:

x = np.arange(start = 0, stop = 10, step = 2)

print(x)

[0 2 4 6 8]

x = np.arange(0, 10, 2)

print(x)

[0 2 4 6 8]

x = np.linspace(start = 0, stop = 10, num = 50)

Basic computations¶

We will cover basics and examples here but there are hundreds of Numpy possibilities - you will have to Google!

https://numpy.org/doc/stable/reference/index.html

x = np.array([[3, 4], [1, 2]])

print(x)

[[3 4]
 [1 2]]

x + 1

array([[4, 5],
       [2, 3]])

y = np.array([[0, 1], [0, 3]])

x + y

array([[3, 5],
       [1, 5]])

x * y

array([[0, 4],
       [0, 6]])

x @ y

array([[ 0, 15],
       [ 0,  7]])

print(x)

[[3 4]
 [1 2]]

np.sum(x)

10

np.sum(x, axis = 0)

array([4, 6])


np.mean(x, axis = 1)

array([3.5, 1.5])

Concatenating & reshaping arrays¶

y = np.concatenate((x, x), axis = 1)

y

array([[3, 4, 3, 4],
       [1, 2, 1, 2]])

y = np.concatenate((x, x), axis = 0)

y

array([[3, 4],
       [1, 2],
       [3, 4],
       [1, 2]])

np.hstack((x, x))
np.vstack((x, x))

array([[3, 4],
       [1, 2],
       [3, 4],
       [1, 2]])
