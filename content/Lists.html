3.3 - Lists¶

Video: Lists

Video available at https://youtube.com/watch?v=MTG-oek7Pus

What is a list?¶

Sequence of values of any type, mutable

x = [0, 1, 7]

type(x)

list

x = ['a', 'b', 9]

x = [[1, 6], [8, 9]]

import numpy as np

x = np.array([[1, 6], [8, 9]])

Indexing into a list¶

x = [4, 10, 11, 15, 16, 20]

x[:3]

[4, 10, 11]

x[-3:-1]

[15, 16]

Mutability¶

x = [1, 5, 6]
y = x

x[0] = 0

print(y)

[0, 5, 6]

List comprehension

x = [5, 10, 13]

plus_two = []
for elem in x:
  plus_two.append(elem + 2)

plus_two

plus_two = [elem + 2 for elem in x]
